About

Hi, i'm Nikita Tchayka, but you can call me Nick.

I'm an aspiring Machine Learning Software Craftsmanship.
What does that mean?

I'm a big enthusiast of Machine Learning, right now the thing that I love the most is Deep Learning and Neural Networks, but all of Machine Learning looks incredible to me.

Also, I love the Software Craftsmanship movement, I embrace concepts like TDD, SOLID and Clean Code.

I always try to combine all these concepts together.

I also love Functional Progamming, I have different levels of experience in languages like Haskell, Scala, Clojure, F#, Java 8 and Rust. But I love programming with this style in my main language Python.

As for personal computing, I have been a user of Linux for several years and my favorite distro is Archlinux, but I recently switched to Windows 10.

Oh, I also use Emacs with Vim keybindings.
